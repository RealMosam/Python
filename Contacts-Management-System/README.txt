/* STEPS */

NOTE: Download The Imported Tkinter and SQlite3 packages if not already installed

For CMD/terminal users:
1. Open CMD/terminal and Change the location to the Project Folder
    Example: cd C:\Users\User_Name\Desktop\Contacts-Management-System
2. Run the main file
    a. python cms_main.py

For any IDE:
1. Open Project Folder
2. Select Contacts-Management-System
3. Define the Python Environment
4. Run the cms_main.py file

/* Other Info */
I've used SQLite As a DataBase.
You can use any database software to open ContactsDataSource.db and accounts.db
I recommend SQLiteStudio.
